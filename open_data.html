MATLAB Code

This MATLAB toolbox is based on the work of Mark Bakker at TU Delft and was designed to generate custom input files for USGS groundwater models, including MODFLOW, SEAWAT, and MT3DMS. These scripts allow the user to import external data, such as elevation data, into the models, and to automate large number of simulations in a batch environment. The scripts are best used for groundwater modeling sensitivity studies, and for post-processing of USGS-based numerical groundwater models.

Educational Outreach

I developed a hydrology field module for the Delaware Nature Society's DuPont Environmental Education Center in Wilmington, DE. The program focuses on groundwater-surface water interactions and emphasizes the use of technology in the classroom and outdoors. To implement the program, instructors need groundwater wells, two water level sensors, and a tidal pond, creek, or estuary. The outdoor activity and manual (student and instructor copy) and a description of the program can be found in the folder below.
